The input value attribute

The value attribute specifies an initial value for an input field:











value Used to provide default value to text input type
readonly A read-only input field cannot be modified
disabled A disabled input field is unusable and un-clickable. The value of
disabled field is not sent when the form is submitted.
size Used to define the visual size of input area.
It is applicable to text, search, tel, url, email, and password
maxlength Used to restrict the maximum number of characters that can be entered.
min/max Used to restrict minimum and maximum value and can be applied to number and date.
multiple Used with file input type, allows user to uplosd multiple files at once.
pattern Used to define a pattern for input, validated on submit by the browser against the defined pattern
Can be used with text, date, search, url, tel, email, and password.
